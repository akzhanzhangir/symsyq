# symsyq

discord bot for playing music using slash commands

## Installation

Use the docker.

```bash
docker pull ghcr.io/djakish/symsyq:main
```

## About
made using [poise](https://github.com/kangalioo/poise) and [songbird](https://github.com/serenity-rs/songbird)
